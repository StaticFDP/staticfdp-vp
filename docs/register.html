How it works

  1. 1Your registration creates a GitHub / Codeberg Issue with label index-registration.
  2. 2A YAML file is added to registered-indexes/ and the harvest pipeline runs.
  3. 3The VP fetches your index.ttl, counts registered FDPs, and adds your index to the federation graph.
  4. 4Your index appears in federation.ttl and the browse UI within 24 h.
